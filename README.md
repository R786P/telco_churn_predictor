
# 📡 Telco Customer Churn Predictor
Predicting customer churn using tabular data + Random Forest. Built as an end-to-end Junior Data Science project.

## 🎯 Business Problem
Identify high-risk customers early to enable proactive retention campaigns. Focus on **Recall** to minimize missed churners (false negatives cost more than false alarms).

## 📊 Dataset
- **Source**: IBM Telco Customer Churn
- **Size**: 7,043 rows × 21 columns
- **Target**: `Churn` (Yes/No)
- **Link**: [Kaggle](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)
